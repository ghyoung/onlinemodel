欢迎使用湖仓建模工具

这是一个高保真原型演示系统，展示了湖仓建模工具的主要功能和界面设计。

请从左侧菜单选择要查看的功能模块，或直接点击下方功能卡片快速体验。

📋 模型管理

管理已创建的模型和表结构，支持版本管理、导出、在线建模、DDL导入等操作。

🔄 血缘关系

可视化展示模型血缘关系，支持血缘深度控制，血缘路径追踪等功能。

📚 标准字段库

维护标准化的字段定义库，支持字段分类、搜索、推荐等功能。

🛡️ 数据治理

提供智能化的数据治理能力，包括质量检测、规则配置、治理报告等。

📝 文档管理

管理建模文档和协作，支持版本历史、协作状态、评论系统等。
