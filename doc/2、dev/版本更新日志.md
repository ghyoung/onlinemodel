# 版本更新日志

## v1.2.0 (2024-12-XX) - 真实数据库连接同步

### 新增功能
- 数据表列表展示组件 `DataSourceTables.tsx`
- 真实数据库连接器 `DatabaseConnector`
- 支持 MySQL、PostgreSQL、MongoDB

### 修复问题
- 同步表结构显示空列表问题
- 表字段查看400错误
- 编辑数据源密码必填问题
- 连接参数必填验证问题

### 重要变更
- 移除所有示例表和模拟数据
- 同步失败时直接返回错误，不创建假数据
- 新增 `ordinal_position` 字段支持
- 实现软删除和级联删除
